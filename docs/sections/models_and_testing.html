Models and Testing

Model testing applies a previously trained BEI classifier to a held-out test set and produces a standardized set of outputs: a per-subject results table plus a sequence of model-evaluation plots. The plots and table are generated directly from the testing scripts so every reported value is traceable to the loaded model file and the processed test subjects.

Select Model

A BEI model file must contain the variables B_final, mu_train, sigma_train, threshold_final, midGap, and roc_cv. These parameters define the classifier, normalization, diagnostic threshold, safety margin threshold, and cross-validated ROC summary.

Once loaded, the Testing menu becomes available for running inference on a test set that contains one or more diagnostic cohorts.

Testing outputs

After testing completes, the toolbox provides two categories of outputs.

1) BEI Test Results table. A sortable table is displayed with one row per subject and the columns Subject, TrueCategory, PredictedCategory, BEI_Score, EED, CouplingRatio, VarianceRatio, LogLikelihood, and BS (see showTestResultsTable.m).

2) Model evaluation plots. A standard plot set is generated to summarize discrimination, calibration, class composition, cohort separation, and likelihood ratios.

Model evaluation plots

The testing pipeline generates the following figures.

  • Population BEI across cohorts (plotPopulationBEI.m): box plots with per-subject scatter overlays and nonparametric significance annotations.
  • ROC curve and calibration (plotROCandCalibration.m): ROC with AUC plus a reliability curve with Brier score and calibration slope and intercept.
  • Clinical zones histogram (plotClinicalZones.m): score distribution histogram with diagnostic and safety thresholds to visualize Disease, At-Risk, and Healthy zones.
  • Test set composition (plotCompositionDiagnosis.m): horizontal bar chart reporting counts and percentages per cohort in the test set.
  • Confusion matrix (plotConfusionMatrix.m): confusion matrix with accuracy, sensitivity, and specificity.
  • Forest plot of likelihood ratios (plotForestPlot.m): per-cohort log10(LR+) and log10(LR−) with 95% confidence intervals.
