Getting started

This section describes the recommended workflow in the order you typically use the BEI Toolbox menus. Menu names may vary slightly by version, but the state gates and dependencies are the same.

Step 1. Load datasets

  • Use the dataset loading menu to register one or more datasets.
  • Confirm each dataset appears in the GUI and the info panel shows dataset name, modality, and subject count.
Loading a dataset does not imply it is preprocessed. It only registers the dataset and its subject list.

Step 2. Preprocess datasets

  • Open the Preprocessing menu.
  • Select a dataset and run preprocessing.
  • After completion, the preprocessing registry marks it as Preprocessed, and downstream menus enable accordingly.

Step 3. Inspect subject-level plots

  • Open Plot or Metrics menu and navigate to subject-level analysis.
  • Inspect energy PDF and covariance summaries for multiple subjects.
  • Use these plots as quality control and to validate assumptions before population inference.

Step 4. Create a study with 2 or more datasets

  • Use the Study menu to define a study by selecting two or more datasets.
  • Ensure study subject counts reflect the included datasets plus any internal datasets (if enabled).
Important: population-level plots and model evaluation are designed around studies because studies define the cohort and grouping logic.

Step 5. Compute energy metrics for the study

  • Run energy metrics computation at the study level.
  • This populates study-level derived outputs used for population plots and optional model workflows.
  • The energy registry marks the study as complete for those metrics.

Step 6. Generate population-level plots

  • Open population metrics plots.
  • Choose a study, then choose metrics such as LL, EED, coupling, variance-related measures, and brain stability.
  • Confirm group comparisons use correct diagnosis labels and include internal healthy controls if configured.

Step 7. Select a model, test, and evaluate

  • Select Model becomes enabled after prerequisites are met (typically preprocessing completion, and energy metrics readiness).
  • Load a model file that contains required variables (example: normalization parameters and ROC data).
  • Use Test Model on a dataset or a study.
  • Use model evaluation plots to calculate BEI scores, view ROC, calibration, and scatter summaries.
