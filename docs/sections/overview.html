Overview

The BEI Toolbox is a MATLAB-based workflow for analyzing brain imaging signals using energy-derived distributions and statistics. It is designed to support cross-dataset and cross-modality analysis while enforcing reproducible processing states through registries.

Scope

  • Supported modalities: EEG, MEG, and fMRI (through EEGLAB, FieldTrip, and SPM pipelines).
  • Supported analysis levels: subject-level visualization and population-level statistics.
  • Model evaluation: loading a pre-trained model and testing it on datasets or studies.

Key data products

  • PDI (Product Data Input): dataset-level preprocessing output containing patient_data.
  • PDO (Product Data Output): study-level energy metrics output containing outputest.
  • Testing results: BEI_TestResults table saved per study under testing_store/.
Design principle: BEI Toolbox is pointer-based. Datasets and studies organize where data live without copying raw recordings. Registries track processing state (preprocessed, metrics computed, tested), and derived products are saved explicitly as PDI, PDO, and testing results.

Scientific grounding of the energy view

BEI Toolbox treats windowed energy correlates as empirical distributions that can be compared across channels, regions, and cohorts. This allows measurement of properties such as distributional form, equality of energy allocation across regions, coupling behavior, variance structure and stability.

What BEI Toolbox does not do

  • It does not replace clinical diagnosis or clinical interpretation.
  • It does not automatically guarantee artifact-free signals; it provides pipelines and quality control visualization.
  • It does not train models (before version 2.0) by default unless you explicitly run the training workflow outside the GUI and save a model file.
Terminology note: energy here refers to an energy-like signal measure derived from the recorded neural correlate. It is a consistent operational definition for analysis and comparison, not a direct measurement of biophysical metabolic energy.
