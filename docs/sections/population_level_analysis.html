Population-Level Analysis

Population-level analysis compares BEI-relevant properties across cohorts using the latest versioned PDO output produced by Energy Metrics. Each population plot is generated from a single PDO*_v*.mat file that contains an outputest struct array.

Inputs

Population analysis depends on:

  • Study PDI snapshot: defines cohort membership, labels, and provenance.
  • Energy artifacts: PDFs, covariance structures, or derived features computed at study scope.
  • Metric definitions: mathematical specification of each summary statistic.
Source-of-truth rule

Subject counts and group membership should be computed from the resolved PDI, including internal controls, not from the currently loaded dataset list.

Population metrics

The toolbox commonly reports a family of scalar population metrics derived from energy distribution properties and coupling proxies. The exact set depends on configuration, but the scientific pattern is stable:

Allocation and EED evidence

Quantifies whether regions share a common energy allocation distribution under a nonparametric test family.

Coupling and complexity proxies

Summaries derived from covariance or state-space representations that reflect network coordination and decoupling.

Population metrics should be defined on fixed estimands and reported with uncertainty summaries where possible (for example bootstrap confidence intervals or cross-validation variability, depending on the metric).

Statistical testing

Energy-derived samples are often non-Gaussian, motivating nonparametric testing. A common workflow is:

  1. Use a Kruskal-Wallis test to detect group differences across diagnoses.
  2. If significant, apply a multiple-comparison post hoc procedure (for example Dunn’s test) to locate pairwise differences.
  3. Report effect sizes alongside p-values, because large cohorts can make small effects statistically significant.

Visualization conventions

The toolbox plotting layer should follow conventions that support scientific reading:

  • Group ordering: a consistent order such as HC, ADHD, FTD, AD to support cross-figure comparison.
  • Overlay strategy: if healthy controls exist in multiple modalities, show them as separate overlays where relevant.
  • Annotation discipline: significance markers reflect corrected post hoc tests, not raw p-values.
  • Denominators visible: show subject counts per group directly on plots or in adjacent tables.

Interpretation

Interpretation should remain tied to the BEI framework:

  • Evidence for equal energy allocation is distributional, not a claim that every time point has identical energy.
  • Coupling proxies should be interpreted relative to preprocessing assumptions and modality measurement characteristics.
  • Cross-modality comparisons should separate physiological statements from measurement statements and unit conventions.
