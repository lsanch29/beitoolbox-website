Brain Entropy Index (BEI) Toolbox

Developed at Johns Hopkins University
Biomedical Engineering / Neuro-medical Control Systems Lab by Luis A. Sanchez Colon

The Brain Entropy Index (BEI) Toolbox is a modular MATLAB-based framework for analyzing multimodal brain imaging data (EEG, MEG, fMRI, iEEG) using maximum entropy and energy-based metrics.
