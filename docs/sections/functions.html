Functions

This page summarizes the primary MATLAB functions and GUI callbacks that implement the BEI Toolbox workflow. Names are shown exactly as they appear in the codebase.

Core BEI functions

FunctionPurpose
define_datasetDefine and register a new dataset from raw data.
defineCohortCallbackSelect datasets and define a cohort (study).
runPreprocessingRun preprocessing pipeline for selected dataset.
energyMetricsCallbackCompute energy-based features for a dataset or study.
buildPreProcessingMenuDynamically build the preprocessing menu in the GUI.
updateInfoPanelUpdate dataset/study information in the GUI panel.
save_patient_dataSave patient-level data structures to the PDI folder.
Tip: If you are reading the code, search for each function name to see where it is called from the GUI menus and registry logic.
