Brain Entropy Index (BEI) Toolbox

A modular MATLAB framework for analyzing multimodal brain imaging data (EEG, MEG, fMRI, iEEG) using maximum entropy and energy based metrics.

Download Read the documentation
Registry driven workflow Subject and population analysis Reproducible PDI and PDO products Model testing and evaluation plots

Quick start

  1. Download the latest release.
  2. Unzip and add the toolbox folder to the MATLAB path.
  3. Launch the GUI and follow the end to end workflow.

Testing and evaluation

Reproducibility by design

BEI Toolbox separates lightweight pointers (Datasets, Studies) from processing state (registries) and from derived products (PDI, PDO, test results). This supports traceable, repeatable workflows.
