Install

  1. Download the release zip and unzip it to a local folder.
  2. In MATLAB, add the toolbox folder to your path.
  3. Launch the GUI and follow the workflow in the documentation.
% Example: add BEI Toolbox to the MATLAB path (edit the folder)
addpath(genpath('/path/to/BEItoolbox'));
savepath;

Then open Getting started for the first end to end run.
